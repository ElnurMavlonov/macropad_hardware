## Project Structure

- `CAD/`: STEP file of the assembled model.
- `PCB/`: KiCad project files (schematic, PCB layout).
- `Firmware/`: (Pending) Firmware source code.
- `production/`: Exported Gerbers and STEP files for case parts.
